# Places Archive

Store records about homes, schools, workplaces, neighborhoods, cities, travel locations, and recurring environments.

Use `LOCATION_SCHEMA.md` and `templates/place-template.md`.

Place records should include:
- known date ranges
- certainty
- associated people
- associated incidents
- routines connected to the place
- sensory details only when sourced
- unresolved questions

Places often anchor timeline reconstruction.
