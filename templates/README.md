# Templates

This directory contains reusable Markdown templates for archive records.

Copy a template into the appropriate archive folder, then complete it using source material and the relevant schema.

Templates are working forms. Do not treat blank fields as evidence.

Available templates:
- `incident-template.md`
- `timeline-template.md`
- `relationship-template.md`
- `era-template.md`
- `autobiography-template.md`
- `place-template.md`
- `digital-life-template.md`
- `personal-profile-template.md`
